### Block states
See also: Block states

Java Edition:

| Name | Default value | Allowed values | Description                              |
|------|---------------|----------------|------------------------------------------|
| axis | `y`           | `x`            | The log or stem is oriented east–west.   |
|      |               | `y`            | The log or stem is oriented vertically.  |
|      |               | `z`            | The log or stem is oriented north–south. |
